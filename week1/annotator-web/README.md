# Week 1 Web Annotator

Modern light-theme wound annotation tool built with React and Tailwind CSS.

## Features

- Load local wound images in browser
- Draw bounding boxes with click and drag
- Assign wound type, action region, severity, and notes
- Navigate images with Prev and Save and Next
- Autosave annotation state in browser local storage
- Export annotations to CSV and COCO-style JSON

## Run

```bash
cd week1/annotator-web
npm install
npm run dev
```

Open the local URL shown in terminal.
